# GMAT Critical Reasoning Practice

This repository tracks my progress in mastering critical reasoning using the **Critical Reasoning Tutor** framework.

## Framework

Every article is analyzed through this sequence:

1. **Conclusion** — What claim is the argument defending?
2. **Premises** — What evidence and reasoning is offered?
3. **Assumptions (Good / True / Happen)** — What hidden values, definitions, and causal chains are assumed?
4. **Gap Test** — What must be true for the premises to support the conclusion?
5. **Weakening** — How can confidence in the conclusion be reduced?
6. **Most Vulnerable Assumption** — Which assumption is the weakest link?
7. **Failure Modes** — Which reasoning errors does the argument commit?

## Structure

Each `article-NN/` folder contains:

| File | Description |
|------|-------------|
| `article.md` | The original article text |
| `analysis.md` | Complete tutor breakdown using the framework |
| `response.md` | My own analysis attempt |
| `evaluation.md` | Tutor's evaluation and scoring of my response |
